2. Core Products & APIs

What are Pangolin's three main product offerings and their core functions?
Pangolin's products can be used individually or in combination, including:
  • Scrape API - Data interface designed for technical teams, providing stable and efficient structured data acquisition (Collection)
  • Data Tracker (AMZ Data Tracker) - Automated data tracking tool accessible to business teams (Tracking), supporting no-code operations and visual configuration
  • DataScaler (Growth Scanner) - Fully automated "growth scanner" (Insights), dedicated to transforming complex web data into actionable business insights
What are the core advantages and capabilities of Scrape API for Amazon data collection?
Scrape API provides stable and efficient data interface services with the following main advantages:
  • Comprehensive Coverage - Covers the entire Amazon platform, including product details, rankings, ad placements, reviews, keywords, and all data types
  • High Timeliness - Achieves minute-level data updates at the fastest
  • Ad Placement Precision - Achieves 98% sponsored ad coverage and supports zip code-level collection
  • High Concurrency Support - Supports unlimited high-concurrency calls
What Google Search Results (SERP) and AI data interfaces does Pangolin provide?
Pangolin provides multiple SERP-related APIs:
  • SERP API - Provides search results returned by search engines for keywords, assisting in product selection. This series includes SERP API, SERP Async API, and SERP Batch API
  • AI Overview SERP API - Exclusive Support for scraping, extracting, and parsing Google AI Overview module content included in some search results, helping sellers understand real user intent
  • Keyword Trends API - Used to obtain keyword popularity trends across different time periods and regions, supporting cross-regional comparison and historical fluctuation analysis
What is the workflow for using AMZ Data Tracker?
The core workflow of AMZ Data Tracker is very simple:
Select Target → Create Tracker → View Results
Users can graphically define target sites, fields, and collection cycles, with support for automated execution at minute, hour, and day levels. Target objects can include:
  • ASIN (Product Identifier)
  • Keywords
  • Category Rankings
  • New Release Rankings
  • Stores
  • Best Seller Rankings

3. Authentication & Billing Rules

How do users start using Pangolin's API and obtain authentication credentials?
Users need to register to obtain a Token. The authentication interface URL is:
POST https://scrapeapi.pangolinfo.com/api/v1/auth
Users need to provide their registered email and password as request parameters. The Token returned by this authentication is the credential for accessing the task submission API and is valid long-term.
What are the validity rules for Pangolin credits? Can unused credits be rolled over to the next month?
Credits are issued on a monthly basis, whether for monthly or annual payments. However, credits are only valid for the current month.
⚠️ Unused credits at the end of the month will be automatically cleared, cannot be rolled over to the next month, and cannot be overdrawn.
What payment methods and discounts does Pangolin offer?
Pangolin provides two main payment methods:
  • Monthly Subscription - Automatically renews monthly, can be canceled anytime
  • Annual Subscription - One-time payment for 12 months, enjoy 20% OFF
